# Interactive Desktop Image Gallery 🖼️
## 🚀 About the Project
This project is a fully functional Desktop Image Gallery developed using **Java (Swing)**. It was designed and built as part of my remote internship at **Code Alpha**. The application demonstrates a clean user interface, dynamic category filtering, and an interactive lightbox feature to provide a seamless photo viewing experience.

## ✨ Key Features
* **Dynamic Filtering:** Easily sort and view images by specific categories (e.g., All, Nature, City, Tech).
* **Responsive Layout:** The grid layout automatically adjusts the flow of thumbnails based on the application window size.
* **Interactive Lightbox Modal:** Click on any thumbnail to open a high-resolution view in a custom modal window.
* **Seamless Navigation:** Use the built-in 'Next' and 'Previous' buttons within the lightbox to browse through the currently filtered category.
* **Hover Animations:** Visual feedback with border highlights when interacting with image thumbnails.
